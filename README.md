# 基于 nodejs 的[立马理财](https://www.lmlc.com)爬虫项目

运行方式
```
npm run user  // 爬取立马理财首页的用户购买信息ajax接口
node run product  // 爬取立马理财理财页所有产品的ajax接口
node run record  // 爬取立马理财产品详情页的投资记录同步页面数据
```
